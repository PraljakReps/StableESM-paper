# CLAUDE.md

## Project Overview

**StableESM Paper** — LaTeX manuscript for "StableESM: Aligning Protein Language Models to Stability Preferences using 1M+ Experimental Mutant Effects"

Workshop paper presented at NeurIPS 2025 (2nd Workshop on Multi-modal Foundation Models and LLMs for Life Sciences). This repo contains the LaTeX source, figures, and bibliography for the manuscript.

The code repository lives at: https://github.com/PraljakReps/StableESM

```bash

pdflatex main.tex
bibtex main
pdflatex main.tex
pdflatex main.tex

# Or with latexmk (recommended)
latexmk -pdf main.tex

# Clean auxiliary files
latexmk -C
```

## Writing Conventions

- **One sentence per line** in `.tex` files for cleaner git diffs
- Use `\citet{}` for inline citations ("Praljak et al. (2025)") and `\citep{}` for parenthetical ("(Praljak et al., 2025)")
- BibTeX keys follow `LastnameYEAR` format (e.g., `Praljak2025`, `Lin2023`)
- Figures are referenced as `\ref{fig:name}`, tables as `\ref{tab:name}`, sections as `\ref{sec:name}`
- Equations use `\label{eq:name}` and are referenced with `\eqref{eq:name}`
- Keep figure files in `figures/` with descriptive names (e.g., `proteingym_spearman_barplot.pdf`)

## Key Technical Terms (consistent usage)

- **StableESM** (not Stable-ESM, stableESM, or stable ESM)
- **ESM-2** (hyphenated, capital)
- **DPO** — Direct Preference Optimization (define on first use)
- **LoRA** — Low-Rank Adaptation (define on first use)
- **ProteinGym** (one word, camel case)
- **MegaScale**, **FireProtDB**, **Human Domainome** — dataset names
- **pLDDT** (lowercase p, uppercase LDDT)
- **$\Delta\Delta G$** for stability changes (not ddG or ΔΔG in plain text)

## Editing Guidelines

- When editing prose, preserve my casual direct voice — don't over-formalize
- Prefer minimal-intervention edits: fix what's wrong, don't rewrite what's fine
- Keep sentences concise; avoid filler phrases ("It is worth noting that...", "Importantly,...")
- When adding new content, match the tone and density of surrounding text
- Don't add packages to the preamble without asking

## Figure Workflow

- Source figures are generated by notebooks in the code repo (`StableESM/notebooks/`)
- Paper-ready versions go in `figures/` as PDF (vector) or high-res PNG (600 DPI)
- Figure captions go in `main.tex`, not in the figure files
